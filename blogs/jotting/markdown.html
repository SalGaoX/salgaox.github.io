    # Markdown基本语法

    # 一、标题

    前面#+空格 1个#代表一个级别，最多支持六级标题 一个#是一级标题，二个#是二级标题，以此类推。支持六级标题。

    示例：

    # 一级标题

    # 二级标题

    # 三级标题

    # 四级标题

    # 五级标题
    # 六级标题

    # 二、字体

    加粗 左右双*号

    斜体 左右单*号

    加粗斜体 左右三*号（粗体两个星号 +斜体一个星号）

    删除线 左右双~号

    # 三、引用

    在需要引用文字前加>号

    示例：

    即将要引用

    这是引用的文字

    # 四、分割线

    三***或者三---号（含三个或以上）

    示例：

    # 五、图片

    语法：![alt](地址 'title') （[alt显示为图片下面的文字](地址为网址全路径，“tittle是图片标题（可选）”

    示例：

    # 六、超链接

    语法：[超链接名称](超链接地址 "超链接title") tittle可选

    示例：

    百度 (opens new window)

    # 七、列表

    语法：无序列表 - + * 号加上空格

    示例：

    • 列表
    • 列表
    • 列表

    语法：有序列表 数字+.+空格+内容 1. 列表1 2. 列表2

    示例：

    1. 这是列表1
    2. 这是列表2

    列表嵌套：上下级之间增加4个空格 或者一个tab

    - 一级列表
    - 二级列表
    1
    2

    示例：

    • 一级列表
      • 二级列表
      • 二级列表2
        • 三级列表
        • 三级列表3

    # 八、表格

    语法：

    表头|表头|表头
--|:--:|--:
内容|内容|内容
内容|内容|内容

第二行用来分割表头和内容，-符号要有两个 原生只要一个
文字默认居左
--两边加：表示文字居中
--右边加：表示文字居右
内容和|之间的空格会被忽略

    # 九、代码

    语法：单行代码用左右反引号`

    `print("这是单行代码")`
    1

    示例：

    ```

import os
```
